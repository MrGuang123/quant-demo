## 🐛 调试技巧

### 常见问题排查

1. **导入错误**
   ```bash
   # 确保在项目根目录
   cd quant-demo
   # 确保虚拟环境激活
   source .venv/bin/activate
   ```

2. **数据获取失败**
   ```python
   # 使用缓存
   df = fetcher.fetch_ohlcv(..., use_cache=True)
   # 减少数据量
   df = fetcher.fetch_ohlcv(..., limit=100)
   ```

3. **回测速度慢**
   - 减少数据量
   - 使用更大的时间周期
   - 减少参数优化的搜索空间
