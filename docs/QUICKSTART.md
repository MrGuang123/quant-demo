# 🚀 快速开始指南

这是一个5分钟快速上手的指南，帮助你快速运行第一个量化交易回测。

---

## 第1步：安装依赖（2分钟）

```bash
# 确保你在项目目录下
cd quant-demo

# 创建虚拟环境（如果还没有）
python3.12 -m venv .venv

# 激活虚拟环境
source .venv/bin/activate  # macOS/Linux
# 或
.venv\Scripts\activate     # Windows

# 安装依赖
pip install --upgrade pip
pip install -r requirements.txt
```

等待安装完成...

## 🆘 遇到问题？

### 常见错误及解决方案

**错误1：ModuleNotFoundError**
```bash
# 解决：确保在虚拟环境中并已安装依赖
source .venv/bin/activate
pip install -r requirements.txt
```

**错误2：网络连接失败**
```bash
# 解决：检查网络，或使用缓存的数据
# 修改 config.py: CACHE_ENABLED = True
```

**错误3：数据获取慢**
```bash
# 解决：使用缓存，减少数据量
# 修改 config.py: DATA_LIMIT = 500
```
